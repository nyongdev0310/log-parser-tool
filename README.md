# Log Parser & Analyzer

A tool that reads log or text files and extracts useful information.

## What it does
- Read log files line by line
- Detect and group `[INFO]`, `[WARN]`, `[ERROR]`
- Filter by date, keyword, or level
- Export results to CSV or Excel

## Tech
- C# / .NET
- Regex
- File IO

> Sample project for text/log parsing and data extraction.
